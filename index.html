Circle Dodger

by: Jack

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I thought it was fairly simple to create. However, it still has lots of different features. I didn't make other, more complex games because I didn't want to spend a lot of time on this, or have the game take lots of time to play.

2. What I Learned

I learned that my game was very fast paced and didn't take lots of time. I also realized there was no specific goal, so I added the goal of staying alive as long as possible.

3. Architecture of My Game

In my game, there are a few main classes being the Player, Circle, CircleManager, and the Panel. The panel controls the player, which just simply moves around. The panel also controls the CircleManager, which controls all the circles. It updates, draws, creates, and deletes all the circles automatically. The panel is what the Player and CircleManager are drawing onto.

4. Technical Implementation

On technical part of my game is detecting when the player touches a red circle. I checked this by using the distance between the center of the player and a circle, and if the distance was less than both the player's radius and the circle's radius combined, they were touching, otherwise, they weren't.

5. Most Challenging Part

The most challenging part of this project was trying to figure out a way to tell if two circles were colliding. At first I tried to see if there was a default circle object in java I could use to detect if it was touching another circle, but that didn't exist. Then, I realized I could use the distance between and radius of both circles to do this.

6. Future Fixes and Updates

I would like to make it so that you could somehow tell where new circles were going to spawn before they spawned, something like an indicator where the new circle is going to be created. I would improve this game by adding different difficulties, so that you could choose how hard each round would be.
